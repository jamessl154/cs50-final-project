Welcome to MyPortfolios!


A simple web app for visualising portfolio progress

The main purpose of the app is to help the user to better visualise their portfolio performance with a focus on relative performance between share purchases. This should help the user to make better decisions for their portfolio in future. The user can also explore by testing hypothetical portfolios. The app uses the IEX cloud API to fetch current and historical prices of shares. It uses those prices to display portfolio performance dynamically to the user.
